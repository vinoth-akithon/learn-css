color --> used to define the color of the text. (<style="color:black;"></style>)
background-color --> used to define the background color of the element. (<style="background-color:green;"></style>)
background-image --> specify the image as background (<style="background-image: url(image-link);"></style>)
background-repeat --> specify the repitation properties.defaultly repeatable (<style="background-repeat:no-repeat;"></style>)
background-position --> specify the position to the image be placed. (<style= "background-position:top right;"></style>)
background-attachment --> specify the image may scroll or fixed (<style "background-attachment:fixed;"></style>)
background-clip --> specify how far the background color extend the element.(<style="background-clip:border-box;">)
background-origin --> specify how far the background image extent the element.(<style="background-origin:centent-box;>)
border-style --> specify the style of the border (<style="border-style:solid">)
border-width --> for specifying the width of the border. (<style="border-width:20px">)
border-color --> to select the border color (<style="border-color:red">)
margin --> used to create a space around elements (<style= "margin:20px;">)
width --> used to specify the width of the element's content area. (<style="width=200px;">)
height --> to set the hight osf the element(content area) (<style="height:200px;>)
padding --> used to create a space around element's content. (<style="padding: 10px 30px 60px 100px">)
outline --> used to draw a line around border. (<style="outline: thin solid #f00>)
outline-offset --> used to add the space between outline and border,which is transparent. (<style="outline:thick groove #ff0; outline-offset: 25px">)
text-align --> used to set the horizontal alignment of a text. (<style="text-align:justify;">)
text-decoration-line --> used to draw a line for text (style="text-decoration:underline;")
text-decoration-style --> to decorate the decoration line (style="text-decoration-style:dotted;")
text-decoration-color --> to specify the color of the decoration line (style="text-decoration-color:red")
text-decoration-thickness --> to specify the width of the decoration line (style="text-decoration-thickness:5px")
text-transform --> used to transform the text to specific case. (style="text-transform:uppercase;")
text-indent --> used to indent the first line of the text (<style="text-indent:25px;">)
letter-spacing --> used to increase or decrease the spacing between letters (<style="letter-spacing:5px">)
line-height --> used to increase or decrease the height of the lines (<style="line-height:20px;">)
word-spacing --> used to increase or decrease the spacing between words. (<style="word-spacing:20px;">)
white-space --> used to set how the white space handle inside the element. (<style= "white-space:pre-wrap">)
font-family --> used to specify the font family link courier or arial (<style="font-family:verdana,sans-serif">)
font-style --> used to style the font like italic (<style="font-style:italic;">)
font-weight --> used to specify the weight of the font (<style="font-weight:bold;">)
cursor --> to style the cursor (<style="cursor:wait";>)
a:link --> for styling unvisited link 
a:visited --> for stying the visited link
a:hover --> for stying the link when hovering
a:active --> for stying the link while clicking
list-style-type --> to specify the type of marker to be used. (<style="list-style-type:square">)
list-style-position --> to specify where the marhker should present either outside(default) or inside (<style="list-style-position:inside">)
list-style-image --> to include the image as marker (<style="list-style-image:url('mail.gif')";>)
list-style --> shorthand property (<style="square inside url('message.gif')";>)
vertical-align --> to align the content vertically like top or bottom in table cell(defaultly -> middle) (<style="vertical-align:bottom">)
border-collapse --> used to collapse the cell borders in table. (<style="border-collapse:collapse;">)
display --> used to convert the display property. (<style="display:flex">)
max-width --> used to set the max width of the element.By using this we can acheive responsive while by using "width" not.
